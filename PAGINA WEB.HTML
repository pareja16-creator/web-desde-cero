Creando una pagina web sencilla

En esta evalucion se tiene el proposito de implementar los conocimientos y de desarollar las habilidades para crear una pagina web sencilla con codigos basicos como HTML Y CSS con estos codigos podremos desarrollar o crear una pagina web desde cero.

¿QUE ES UNA PAGINA WEB?

Se conoce como página web, página electrónica o página digital a un documento digital de carácter multimediático (es decir, capaz de incluir audio, video, texto y sus combinaciones), adaptado a los estándares de la World Wide Web (WWW) y a la que se puede acceder a través de un navegador web y una conexión activa a Internet. Se trata del formato básico de contenidos en la red.

Las páginas web se encuentran programadas en un formato HTML o XHTML, y se caracterizan por su relación entre unas y otras a través de hipervínculos: enlaces hacia contenidos diversos que permiten una lectura compleja, simultánea y diversa, muy distinta a la que podemos hallar en los libros y revistas.

¿Para que sirve una pagina web y cual es su proposito?

Las páginas web cumplen básicamente con la tarea de brindar información de cualquier índole y en cualquier estilo o grado de formalidad. Algunas, al mismo tiempo, permiten distintos grados de interacción entre usuarios o con alguna institución, como son las páginas de foros, servicios de citas o redes sociales, las páginas de compra y venta de bienes, las páginas de consulta o de contacto con empresas, instituciones gubernamentales o con ONGs, e incluso las páginas de soporte técnico especializado. En principio, las funciones de una página web son tan amplias como la demanda de los usuarios y la oferta de sus creadores.

El propósito de una página web es Cumplir las necesidades de la audiencia. Alcanzar objetivos específicos. Proporcionar una experiencia óptima. Comunicar la marca. Mejorar la visibilidad en línea. Generar respuesta y acciones medibles. Presentar información, facilitar el contacto y la comunicación, ayudar al marketing y promoción, impulsar el comercio electrónico, aumentar la generación de leads, brindar una experiencia de usuario, ofrecer análisis de datos y contenido actualizado.

CODIGOS BASICOS DE UNA PAGINA WEB

1. HTML (HyperText Markup Language): HTML es el lenguaje de marcado estándar utilizado para crear la estructura de una página web. Define los elementos y su organización, como títulos, párrafos, enlaces e imágenes.

2. CSS (Cascading Style Sheets): CSS se encarga de definir el diseño y la presentación de los elementos HTML. Permite personalizar colores, fuentes, márgenes y crear diseños responsivos para adaptarse a diferentes dispositivos.

3. JavaScript: JavaScript es un lenguaje de programación que permite añadir interactividad a una página web. Con JavaScript, puedes crear efectos visuales, validar formularios, cargar contenido dinámicamente y mucho más
